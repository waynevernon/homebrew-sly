# homebrew-sly

Homebrew tap for [Sly](https://github.com/waynevernon/sly).

## Install

Install the stable channel:

```bash
brew tap waynevernon/sly
brew install --cask waynevernon/sly/sly
```

Install the beta channel:

```bash
brew tap waynevernon/sly
brew install --cask waynevernon/sly/sly@beta
```

Stable and beta conflict by design, so install one channel or the other.
